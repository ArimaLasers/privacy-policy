1. 資料收集與使用 / Data Collection and Usage

我們僅處理應用運作所需的基本資料，未主動收集使用者個人資訊。

We only process data necessary for the app to function and do not actively collect personal user information.

1.2 其他資料處理 / Additional Data Processing

藍牙 MAC 位址等設備資訊僅用於即時通訊。
繪圖內容主要儲存在使用者的本地裝置中。
當執行圖形格式轉換時，資料將傳送至應用伺服器進行臨時處理，伺服器不會儲存、分析或與任何用戶識別資料關聯，處理完成後即時返回結果。

Bluetooth MAC addresses and similar device identifiers are used solely for real-time communication.
Drawing content is primarily stored on the user's local device.
During format conversion operations, data is transmitted to the application server for temporary processing. The server does not store, analyze, or associate the data with any user-identifiable information, and the result is returned immediately after processing.

2. 資料安全 / Data Security

  • 藍牙通訊使用 AES-128 加密 / Bluetooth communication is secured with AES-128 encryption
  • 資料僅儲存在應用沙盒內 / All data is stored in the app sandbox
  • 未進行任何雲端同步 / No cloud synchronization is performed

3. 兒童條款 / Children's Privacy

本應用不針對 13 歲以下兒童設計，亦不會主動收集兒童個人資料。若您為家長或監護人，發現兒童使用本應用，請聯絡我們協助刪除資料。
This application is not intended for children under 13 and does not knowingly collect personal information from children. If you are a parent or guardian and believe your child has used the app, please contact us to assist in data removal.

4. 第三方服務 / Third-Party Services

本應用未整合任何第三方資料收集或分析工具（如 Firebase、AdMob 等）。除了特定圖形格式轉換操作外，所有資料處理皆於使用者裝置本地進行。
This app does not integrate any third-party analytics or advertising tools (e.g., Firebase, AdMob). Except for specific format conversion operations, all data is processed locally on the user's device.

5. 用戶權利 / User Rights

  • 可隨時撤回授權權限（如藍牙、定位）
    You may revoke permissions (e.g., Bluetooth, location) via system settings
  • 解除安裝即清除所有資料
    Uninstalling the app deletes all local data
  • 資料請求請聯絡我們
    For data-related requests, contact us

6. 政策更新 / Policy Updates

本政策可能因功能更新或法規要求調整，更新內容將於本頁公布。
This policy may be updated due to feature changes or legal requirements; changes will be published on this page.
